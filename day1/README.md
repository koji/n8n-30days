# Day 1 - Environment Setup

Install Docker Desktop or Orb Stack, then prepare the environment variables:

```shell
openssl rand -hex 32
```

Bring up the stack:

```shell
docker compose up -d
```

### Set up n8n account

Visit http://localhost:5678/setup to finish onboarding and create your n8n account.
